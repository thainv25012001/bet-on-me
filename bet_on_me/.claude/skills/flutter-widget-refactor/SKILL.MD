name: flutter-widget-refactor
description: Refactors large Flutter widgets into clean, reusable components. Use when improving code readability or reducing complex widget trees.
---

When refactoring Flutter widgets, always include:

1. **Start with an analogy**  
   Compare the widget structure to something from everyday life (like organizing rooms in a house).

2. **Draw the current structure**  
   Use ASCII diagrams to show the existing widget tree.

3. **Identify problems**  
   Point out issues such as:
   - widgets that are too large
   - deeply nested layouts
   - duplicated UI logic

4. **Show the improved structure**  
   Provide a cleaner widget breakdown with reusable components.

5. **Highlight a gotcha**  
   Explain common mistakes developers make when structuring widgets.

Focus on modular, readable, and maintainable Flutter UI design.
